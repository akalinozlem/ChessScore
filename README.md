# ChessScore
Calculates the total score of the white and black teams on the chessboard

This code calculates scores for each team based on the state of the chessboard.


Requirements:
1) The txt file with the chessboard information must be in the same directory as the exe file.
2) The name of the txt file containing the chessboard information must start with "board".
2) When viewed in two dimensions, the black team must be at the top of the board; the white team must be at the bottom of the board.


Operation:
When you run ChessScore.exe, it will get the information inside all txt files starting with "board" and a txt file with the total score for each file will be created in the same directory.
